Local Signals Pro — десктопное приложение для автоматического мониторинга криптовалютных сигналов.

Отслеживает 10 популярных монет на Bybit и Binance, анализирует три технических индикатора и отправляет уведомления в Telegram когда большинство индикаторов сходятся в одном направлении. Встроенный фильтр по старшему таймфрейму отсекает сигналы против основного тренда, что повышает качество входов.
